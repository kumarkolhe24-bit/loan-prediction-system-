# 🚀 Quick Start Guide - Loan Decision Intelligence System

## ⚡ 5-Minute Setup

### Step 1: Verify Installation ✅
```bash
# Check if all files are present
dir
```

You should see:
- ✅ loan_approval_dataset.csv
- ✅ loan_intelligence.py
- ✅ phase1-4 scripts
- ✅ backend/ folder
- ✅ frontend/ folder
- ✅ model/ folder (after running phases)
- ✅ data/ folder (after running phases)

### Step 2: Install Dependencies 📦
```bash
pip install -r requirements.txt
```

This installs:
- pandas, numpy, scikit-learn (ML)
- shap (Explainability)
- fastapi, uvicorn (Backend API)
- streamlit, plotly (Frontend UI)

### Step 3: Run Setup (First Time Only) 🔧
```bash
# Run all phases in sequence
python phase1_data_validation.py
python phase2_preprocessing.py
python phase3_model_training.py
python phase4_explainability.py
```

**Expected Output:**
- ✅ Phase 1: Dataset validated (4,269 samples)
- ✅ Phase 2: Features preprocessed (8 features)
- ✅ Phase 3: Model trained (98.48% accuracy)
- ✅ Phase 4: Explainability configured

**Time**: ~2-3 minutes

### Step 4: Test the System 🧪
```bash
python test_system.py
```

**Expected Output:**
```
🧪 COMPREHENSIVE SYSTEM TESTING
✅ TEST PASSED (6/6 tests)
🎉 ALL TESTS PASSED!
```

### Step 5: Start the Application 🎨

#### Option A: Using Batch Files (Windows)
```bash
# Terminal 1: Start Frontend
run_frontend.bat

# Terminal 2: Start Backend (optional)
run_backend.bat
```

#### Option B: Manual Start
```bash
# Terminal 1: Start Frontend
cd frontend
streamlit run app.py

# Terminal 2: Start Backend (optional)
cd backend
python api.py
```

### Step 6: Access the Application 🌐

**Frontend UI**: http://localhost:8501  
**Backend API**: http://localhost:8000  
**API Docs**: http://localhost:8000/docs

## 🎯 Using the System

### 1. Make a Prediction

1. Open http://localhost:8501
2. Fill in the loan application form:
   - Personal info (dependents, education, employment)
   - Financial info (income, loan amount, CIBIL score)
   - Assets info (residential, commercial, luxury, bank)
3. Click "🔍 Analyze Application"
4. View results:
   - ✅ Approved or ❌ Rejected
   - Risk Score (0-100)
   - Feature Impact Chart
   - Suggestions for improvement

### 2. Run What-If Analysis

1. Go to "🔮 What-If Analysis" tab
2. Select scenarios to test:
   - Different CIBIL scores
   - Different income levels
3. Click "Run What-If Analysis"
4. View how changes affect approval chances

### 3. View History

1. Go to "📈 History" tab
2. See all past predictions
3. View statistics and distributions

## 🔧 Troubleshooting

### Issue: "Module not found"
**Solution**: Install dependencies
```bash
pip install -r requirements.txt
```

### Issue: "File not found: model/model.pkl"
**Solution**: Run setup phases
```bash
python phase2_preprocessing.py
python phase3_model_training.py
```

### Issue: "Port already in use"
**Solution**: Change port or kill existing process
```bash
# For Streamlit (default 8501)
streamlit run app.py --server.port 8502

# For FastAPI (default 8000)
# Edit backend/api.py, change port in uvicorn.run()
```

### Issue: Streamlit asks for email
**Solution**: Config file already created at `frontend/.streamlit/config.toml`

### Issue: Warnings about feature names
**Solution**: These are harmless sklearn warnings, can be ignored
